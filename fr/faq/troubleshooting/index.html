    Dépannage

    Problèmes de téléchargement

    Symptôme: "Le téléchargement a échoué"

    1. Vérifiez votre connexion Internet
    2. Assurez-vous que le fichier est inférieur à 50 MB
    3. Vérifiez le format (PDF, DOC, DOCX, MD, TXT)
    4. Videz le cache de votre navigateur
    5. Essayez un autre navigateur

    Erreurs de génération

    Symptôme: "Erreur lors de la génération"

    1. Attendez 30 secondes et réessayez
    2. Réduisez le nombre de questions
    3. Simplifiez le sujet
    4. Vérifiez l'état du système

    Problèmes de performance

    Symptôme: Temps de chargement lents

    1. Fermez les autres onglets du navigateur
    2. Videz le cache de votre navigateur
    3. Mettez à jour votre navigateur
    4. Vérifiez votre vitesse Internet

    Erreurs d'affichage

    Symptôme: Les éléments ne s'affichent pas correctement

    1. Actualisez la page (F5)
    2. Actualisation forcée (Ctrl+Maj+R)
    3. Vérifiez votre version de navigateur
    4. Désactivez les extensions du navigateur

    Review Queue et Exam Composer

    Les questions n'apparaissent pas dans Exam Composer

    Cause: Seules les questions avec le statut «Approuvée» apparaissent dans Exam Composer.

    Solution: 1. Ouvrez la Review Queue 2. Filtrez par statut «En attente» 3. Vérifiez et approuvez les questions souhaitées 4. Elles sont ensuite immédiatement disponibles dans Exam Composer

    Chat de documents

    «Chat de documents» n'est pas visible ou accessible

    Cause: Le Chat de documents est disponible à partir de l'abonnement Starter.

    Solution: Vérifiez votre abonnement actuel sous Abonnement. Cette fonctionnalité n'est pas incluse dans le plan Free.

    Build et documentation

    La compilation mkdocs échoue avec des fichiers manquants

    Cause: Les pages référencées dans mkdocs.yml n'existent pas encore.

    Solution: Assurez-vous que tous les fichiers .md enregistrés dans la navigation existent réellement.

    Support

    Si le problème persiste:

    • E-mail: support@examcraft.ai
    • Temps de réponse: moins de 24 heures (jours ouvrables)
