== System Overview

ExamCraft AI transforms the way educators create exam questions by using artificial intelligence to generate high-quality, contextual questions from course materials.

[cols="1,2"]
|===
|**Technology Stack** | React + TypeScript + FastAPI + PostgreSQL + ChromaDB
|**AI Integration** | Claude API for question generation
|**Deployment** | Docker Compose for development, scalable for production
|**Documentation** | AsciiDoc with PlantUML diagrams
|===

== Key Features

* 🤖 **AI-Powered Generation** - Create questions from your course materials
* 📄 **Document Processing** - Support for PDF, Word, text, and markdown files
* 🔍 **Vector Search** - RAG-based contextual question generation
* 📝 **Multiple Question Types** - Multiple choice and open-ended questions
* 🎚️ **Difficulty Control** - Easy, medium, and hard levels with Bloom's Taxonomy
* 📊 **Review & Approval** - Complete editorial control over generated content
* 📋 **Export Options** - PDF, Word, Moodle XML, and other formats

== Building Documentation

This documentation is built using AsciiDoc and can be generated in multiple formats:

[source,bash]
----
# Install AsciiDoctor
gem install asciidoctor asciidoctor-diagram

# Generate HTML
asciidoctor index.adoc

# Generate PDF
asciidoctor-pdf index.adoc

# Generate with diagrams
asciidoctor -r asciidoctor-diagram index.adoc
----
