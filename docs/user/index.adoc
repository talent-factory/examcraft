== Getting Started

=== What is ExamCraft AI?

ExamCraft AI is an intelligent platform that helps educators automatically generate high-quality exam questions using artificial intelligence. Instead of spending hours creating questions manually, you can:

* 📄 **Upload course materials** (PDFs, documents)
* 🤖 **Generate contextual questions** based on your content
* 📝 **Review and customize** generated questions
* 📊 **Export complete exams** in multiple formats

=== Workflow Steps

==== 1. Document Upload
[source]
----
Upload → Process → Index → Ready
----

* Drag & drop or select files
* Support for PDF, DOC, DOCX, TXT, MD
* Automatic content extraction
* Semantic indexing for search

==== 2. Question Generation
[source]
----
Select Topic → Choose Settings → Generate → Review
----

* Choose from available documents
* Set difficulty level (Easy, Medium, Hard)
* Select question types
* AI generates contextual questions

==== 3. Review & Approval
[source]
----
Review → Edit → Approve → Save
----

* Check question quality
* Edit text and options
* Verify correct answers
* Approve for exam use

==== 4. Exam Creation
[source]
----
Select Questions → Arrange → Export → Download
----

* Choose approved questions
* Organize by topic/difficulty
* Generate formatted exam
* Download in preferred format

== Question Types

=== Multiple Choice Questions

**Format**: One question with 4 answer options

**Example**:
```
What is the primary purpose of Docker containers?

A) To replace virtual machines entirely
B) To provide lightweight application isolation
C) To serve as a database management system
D) To compile source code automatically

Correct Answer: B
Explanation: Docker containers provide lightweight, portable application isolation...
```

=== Open-ended Questions

**Format**: Question requiring detailed written response

**Example**:
```
Explain the difference between supervised and unsupervised learning in machine learning,
and provide one example of each approach.

Sample Answer: Supervised learning uses labeled training data to learn patterns...
Grading Criteria:
• Defines both learning types (25%)
• Explains key differences (25%)
• Provides valid examples (25%)
• Demonstrates understanding (25%)
```

=== Future Question Types

* **True/False** with explanation
* **Fill-in-the-blank** questions
* **Matching** exercises
* **Code completion** for programming courses
* **Image-based** questions

== Best Practices

=== Document Preparation

**Recommended Formats**:

* PDF with clear text (not scanned images)
* Word documents with good structure
* Markdown with proper headings
* Plain text with logical organization

**Content Guidelines**:

* Include clear section headings
* Use consistent terminology
* Provide examples and explanations
* Ensure content completeness

=== Question Review

**Quality Checklist**:

* ✅ Question is clear and unambiguous
* ✅ Correct answer is indisputable
* ✅ Distractors are plausible but incorrect
* ✅ Language is appropriate for audience
* ✅ Content aligns with learning objectives

**Common Issues to Avoid**:

* ❌ Trick questions or gotchas
* ❌ Multiple correct answers
* ❌ Overly complex language
* ❌ Cultural or regional bias
* ❌ Outdated information
