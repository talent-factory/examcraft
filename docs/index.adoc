== Project Overview

ExamCraft AI is a modern web application that helps educators automatically generate high-quality exam questions using artificial intelligence.

=== Key Features

* 🤖 **AI-powered Question Generation** with Claude API
* 📄 **Document-based RAG System** for contextual questions
* 📝 **Multiple Question Types**: Multiple Choice, Open Questions
* 🎚️ **Difficulty Levels**: Easy, Medium, Hard with Bloom Taxonomy
* 🌐 **Multilingual Support**: German and English
* 📊 **Immediate Evaluation** with detailed feedback
* 🐳 **Docker-based Development** for easy setup

=== Technology Stack

[cols="1,2,3"]
|===
|Layer |Technology |Purpose

|**Frontend**
|React 18 + TypeScript + Material-UI
|Modern, responsive user interface

|**Backend**
|FastAPI + Python + PydanticAI
|High-performance API with AI integration

|**Database**
|PostgreSQL + Redis
|Persistent storage and caching

|**AI/ML**
|Claude API + ChromaDB + sentence-transformers
|Question generation and vector search

|**DevOps**
|Docker Compose + nginx
|Containerization and deployment
|===

== Getting Started

=== Prerequisites

* Docker & Docker Compose
* Git
* Claude API Key (optional for demo mode)

=== Quick Setup

[source,bash]
----
# Clone repository
git clone <repository-url>
cd ExamCraft

# Start development environment
./start-dev.sh

# Access application
# Frontend: http://localhost:3000
# Backend: http://localhost:8000/docs
----
