# 📸 ExamCraft AI Screenshots

Dieses Verzeichnis enthält Screenshots der ExamCraft AI Benutzeroberfläche für die Dokumentation.

## 📋 Übersicht

Alle Screenshots sollten folgende Kriterien erfüllen:

- **Format**: PNG (bevorzugt) oder JPG
- **Auflösung**: 1920x1080 oder höher
- **Dateigröße**: <500 KB (komprimiert)
- **Namenskonvention**: `{nummer}_{beschreibung}.png`

## 📝 Verwendung in Dokumentation

Screenshots können in Markdown-Dokumenten wie folgt eingebunden werden:

```markdown
![Beschreibung](screenshots/01_main_navigation.png)
```

Mit Bildunterschrift:

```markdown
![Hauptnavigation](screenshots/01_main_navigation.png)
*Abbildung 1: ExamCraft AI Hauptnavigation*
```

## 🔄 Aktualisierung

Screenshots sollten aktualisiert werden, wenn:

- Größere UI-Änderungen vorgenommen wurden
- Neue Features hinzugefügt wurden
- Das Design überarbeitet wurde
- Fehler in bestehenden Screenshots gefunden wurden
