== System Overview

ExamCraft AI is built as a modern, microservices-oriented web application with a clear separation between frontend, backend, data, and AI services. The architecture prioritizes scalability, maintainability, and performance.

=== Data Layer

==== PostgreSQL Database

**Schema Design**:
[source,sql]
----
-- Core entities
CREATE TABLE documents (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    filename VARCHAR(255) NOT NULL,
    file_path VARCHAR(512) NOT NULL,
    file_size BIGINT NOT NULL,
    mime_type VARCHAR(100),
    status VARCHAR(50) DEFAULT 'uploading',
    metadata JSONB,
    user_id UUID NOT NULL,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE questions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    type VARCHAR(50) NOT NULL,
    question TEXT NOT NULL,
    options JSONB,
    correct_answer TEXT NOT NULL,
    explanation TEXT,
    difficulty VARCHAR(20),
    bloom_level VARCHAR(50),
    source_documents UUID[] DEFAULT '{}',
    metadata JSONB,
    created_at TIMESTAMP DEFAULT NOW(),
    approved_at TIMESTAMP
);

CREATE TABLE exams (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(255) NOT NULL,
    description TEXT,
    question_ids UUID[] NOT NULL,
    settings JSONB,
    created_by UUID NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);
----

==== Redis Cache

**Caching Strategy**:
* API response caching for frequently accessed data
* Session management
* Rate limiting counters
* Temporary data storage

**Cache Keys**:

[source,text]
----
documents:user:{user_id}       # User's documents
questions:document:{doc_id}    # Questions per document
api:rate_limit:{user_id}       # Rate limiting
session:{session_id}           # User sessions
----

==== ChromaDB Vector Storage

**Collection Strategy**:
* One collection per document for isolation
* Metadata-rich storage for filtering
* Persistent local storage
* Efficient similarity search indexing

**Vector Schema**:

[source,json]
----
{
  "id": "doc_{document_id}_chunk_{chunk_index}",
  "embedding": [0.1, -0.2, 0.8, ...],  // 384-dimensional vector
  "metadata": {
    "document_id": "uuid",
    "chunk_index": 0,
    "page_number": 1,
    "section_title": "Introduction",
    "chunk_type": "paragraph"
  },
  "document": "The actual text content..."
}
----

=== Security Measures

==== Input Validation

[source,python]
----
# Pydantic models for request validation
class DocumentUploadRequest(BaseModel):
    filename: str = Field(..., min_length=1, max_length=255)
    file_size: int = Field(..., gt=0, le=50_000_000)  # Max 50MB
    mime_type: str = Field(..., regex=r'^(application|text)/')

class QuestionGenerationRequest(BaseModel):
    topic: str = Field(..., min_length=3, max_length=200)
    document_ids: List[UUID] = Field(..., min_items=1, max_items=10)
    question_count: int = Field(default=5, ge=1, le=20)
    difficulty: Literal['easy', 'medium', 'hard'] = 'medium'
----

==== File Security

* File type validation beyond MIME type
* Virus scanning integration points
* Secure file storage with access controls
* Temporary file cleanup

==== API Security

* Rate limiting per user/IP
* Request size limits
* CORS configuration
* API key management for external services

=== Production Deployment

==== Container Strategy

[source,dockerfile]
----
# Multi-stage build for optimization
FROM node:18-alpine AS frontend-builder
COPY frontend/ .
RUN bun install --frozen-lockfile && bun run build

FROM python:3.13-slim AS backend-builder
COPY backend/ .
RUN pip install --no-cache-dir -r requirements.txt

# Final runtime image
FROM python:3.13-slim
COPY --from=backend-builder /app /app
COPY --from=frontend-builder /app/build /app/static
EXPOSE 8000
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
----

== Monitoring & Observability

=== Health Checks

[source,python]
----
@router.get("/health")
async def health_check():
    """Comprehensive health check"""
    checks = {
        "database": await check_database_connection(),
        "redis": await check_redis_connection(),
        "vector_db": await check_vector_db_connection(),
        "file_storage": await check_file_storage(),
        "claude_api": await check_claude_api_status()
    }

    all_healthy = all(checks.values())
    status_code = 200 if all_healthy else 503

    return JSONResponse(
        status_code=status_code,
        content={
            "status": "healthy" if all_healthy else "unhealthy",
            "checks": checks,
            "timestamp": datetime.utcnow().isoformat()
        }
    )
----
