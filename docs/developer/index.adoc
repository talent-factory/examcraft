== Development Environment

=== System Requirements

* **Docker**: >= 20.10
* **Docker Compose**: >= 2.0
* **Node.js**: >= 18 (for local frontend development)
* **Python**: >= 3.13 (for local backend development)
* **Git**: >= 2.30

=== Local Development Setup

==== 1. Clone and Setup

[source,bash]
----
# Clone repository
git clone <repository-url>
cd ExamCraft

# Copy environment template
cp .env.example .env

# Edit .env and add your Claude API key
nano .env
----

==== 2. Start Development Environment

[source,bash]
----
# Start all services
./start-dev.sh

# Or manually with docker-compose
docker-compose up --build -d
----

==== 3. Verify Installation

[source,bash]
----
# Check services status
docker-compose ps

# Test API health
curl http://localhost:8000/health

# Access applications
# Frontend: http://localhost:3000
# Backend: http://localhost:8000
# API Docs: http://localhost:8000/docs
----

==== Document Processing Pipeline

[source,python]
----
# services/document_service.py
class DocumentService:
    async def process_document(self, file_path: str) -> ProcessedDocument:
        # 1. Extract content with Docling
        content = await self.docling_service.process(file_path)

        # 2. Create semantic chunks
        chunks = self.create_chunks(content.text)

        # 3. Generate embeddings
        embeddings = self.embedding_model.encode(chunks)

        # 4. Store in vector database
        await self.vector_service.store_chunks(
            document_id=content.id,
            chunks=chunks,
            embeddings=embeddings
        )

        return ProcessedDocument(
            id=content.id,
            title=content.title,
            chunks_count=len(chunks),
            status="processed"
        )
----

==== RAG Question Generation

[source,python]
----
# services/rag_service.py
class RAGService:
    async def generate_questions(
        self,
        topic: str,
        document_ids: List[str],
        count: int = 5
    ) -> List[Question]:
        # 1. Retrieve relevant context
        context = await self.vector_service.search_similar(
            query=topic,
            document_ids=document_ids,
            top_k=10
        )

        # 2. Build RAG prompt
        prompt = self.build_rag_prompt(topic, context, count)

        # 3. Generate with Claude
        response = await self.claude_service.generate(prompt)

        # 4. Parse and validate questions
        questions = self.parse_questions(response)

        return questions
----

=== Frontend Technologies

==== React Architecture

[source,typescript]
----
// components/DocumentUpload.tsx
import React, { useCallback } from 'react';
import { useDropzone } from 'react-dropzone';
import { DocumentService } from '../services/DocumentService';

interface DocumentUploadProps {
  onUploadComplete: (documentId: string) => void;
}

export const DocumentUpload: React.FC<DocumentUploadProps> = ({
  onUploadComplete
}) => {
  const onDrop = useCallback(async (files: File[]) => {
    for (const file of files) {
      try {
        const result = await DocumentService.upload(file);
        onUploadComplete(result.documentId);
      } catch (error) {
        console.error('Upload failed:', error);
      }
    }
  }, [onUploadComplete]);

  const { getRootProps, getInputProps, isDragActive } = useDropzone({
    onDrop,
    accept: {
      'application/pdf': ['.pdf'],
      'application/msword': ['.doc'],
      'application/vnd.openxmlformats-officedocument.wordprocessingml.document': ['.docx'],
      'text/plain': ['.txt'],
      'text/markdown': ['.md']
    }
  });

  return (
    <div {...getRootProps()} className="dropzone">
      <input {...getInputProps()} />
      {isDragActive ? (
        <p>Drop files here...</p>
      ) : (
        <p>Drag & drop files, or click to select</p>
      )}
    </div>
  );
};
----

== Debugging & Troubleshooting

=== Common Issues

==== Port Conflicts

[source,bash]
----
# Check what's using port 3000/8000
lsof -i :3000
lsof -i :8000

# Kill processes using ports
sudo kill -9 $(lsof -t -i:3000)
sudo kill -9 $(lsof -t -i:8000)
----

==== Container Issues

[source,bash]
----
# View container logs
