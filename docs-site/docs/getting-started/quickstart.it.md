# Guida di avvio rapido

Create il vostro primo esame con intelligenza artificiale in 5 minuti.

## Prerequisiti

- Browser moderno (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Connessione Internet
- Account ExamCraft AI

## Passaggio 1: Accesso

1. Aprite ExamCraft AI nel vostro browser
2. Accedete con il vostro account Google o e-mail
3. Dopo l'accesso, vedrete il Dashboard

## Passaggio 2: Caricamento documento

1. Fate clic su **Carica documenti** nella navigazione
2. Trascinate un file PDF o Word nell'area di caricamento (oppure fate clic su **Seleziona file**)
3. Attendete il completamento dell'elaborazione (10–45 secondi)

!!! tip "Suggerimento"
    I documenti strutturati con intestazioni chiare generano domande d'esame di migliore qualità.

## Passaggio 3: Creazione esame

1. Passate alla scheda **Crea esame RAG**
2. Selezionate i vostri documenti caricati
3. Configurate:
    - **Argomento/Focus**: ad es. "Algoritmi di ordinamento"
    - **Numero domande**: consigliato 5–10
    - **Tipi domanda**: Scelta multipla e/o domande aperte
    - **Livello difficoltà**: Facile / Medio / Difficile
4. Fate clic su **Genera esame RAG**

## Passaggio 4: Verifica risultato

Dopo 20–60 secondi vedrete le vostre domande generate con:

- Testo domanda e opzioni di risposta
- Risposta corretta con spiegazione
- Riferimenti alle vostre fonti documentali
- Punteggio di confidenza (Confidence Score) – indicatore di qualità
