# Créer un compte

Pour utiliser ExamCraft AI, vous avez besoin d'un compte utilisateur personnel. L'inscription est gratuite et prend moins de deux minutes.

!!! warning "Pas d'auto-inscription dans certaines institutions"
    Dans certaines institutions éducatives, l'auto-inscription est désactivée. Dans ce cas, vous recevrez un e-mail d'invitation de votre institution. Suivez le lien dans cet e-mail pour configurer votre compte. Contactez votre service informatique ou votre administrateur d'institution si vous n'avez pas reçu d'invitation.

## Méthodes d'inscription

ExamCraft AI prend en charge deux procédures de connexion. Vous pouvez choisir l'une des méthodes suivantes à tout moment:

### Google OAuth (recommandé)

La connexion via Google OAuth est la méthode la plus rapide et la plus sécurisée. Vous n'avez besoin d'aucun mot de passe supplémentaire — votre identité Google existante est utilisée pour l'authentification.

!!! tip "Conseil: Préférer Google OAuth"
    Google OAuth est plus sécurisé qu'un mot de passe autonome car Google gère les mécanismes de sécurité modernes comme l'authentification à deux facteurs et la détection des anomalies. Si vous utilisez déjà un compte Google, nous recommandons cette méthode.

**Guide étape par étape:**

1. Ouvrez ExamCraft AI dans votre navigateur et accédez à la page de connexion.
2. Cliquez sur le bouton **Se connecter avec Google**.
3. Une fenêtre de connexion Google s'ouvre (popup ou nouvelle page). Sélectionnez le compte Google souhaité ou entrez votre adresse e-mail Google.
4. Confirmez l'accès lorsque Google vous le demande. ExamCraft AI reçoit uniquement votre nom et votre adresse e-mail — pas de contenu de votre compte Google.
5. Vous êtes automatiquement redirigé vers ExamCraft AI et vous êtes immédiatement connecté.
6. Lors de votre première connexion, votre profil est automatiquement créé. Vous verrez ensuite le Dashboard avec votre abonnement Free.

### E-mail et mot de passe

Si vous ne souhaitez pas ou ne pouvez pas utiliser un compte Google, inscrivez-vous avec votre adresse e-mail et un mot de passe de votre choix.

**Guide étape par étape:**

1. Ouvrez ExamCraft AI dans votre navigateur et accédez à la page de connexion.
2. Cliquez sur **S'inscrire** (ou **Créer un compte**).
3. Entrez votre **adresse e-mail**. Utilisez une adresse e-mail à laquelle vous avez accès régulièrement — elle sera nécessaire pour la confirmation du compte et la réinitialisation du mot de passe.
4. Choisissez un **mot de passe**. Le mot de passe doit satisfaire aux exigences suivantes:
    - Au minimum 8 caractères
    - Au minimum une lettre majuscule
    - Au minimum un chiffre ou un caractère spécial
5. Répétez le mot de passe dans le champ **Confirmer le mot de passe**.
6. Cliquez sur **S'inscrire**.
7. Vous recevrez un **e-mail de confirmation** à l'adresse indiquée. Ouvrez cet e-mail et cliquez sur le lien de confirmation pour activer votre compte. Le lien est valide 24 heures.
8. Après la confirmation, vous pouvez vous connecter avec votre adresse e-mail et votre mot de passe.

!!! note "Vous n'avez pas reçu l'e-mail de confirmation?"
    Commencez par vérifier votre dossier spam ou indésirable. Si l'e-mail n'y apparaît pas, vous pouvez demander une nouvelle confirmation sur la page de connexion sous **Renvoyer l'e-mail de confirmation**.

## Première connexion

Après l'inscription réussie et la première connexion, vous êtes automatiquement redirigé vers le **Dashboard**. Le Dashboard est le point de départ central pour toutes les activités dans ExamCraft AI.

Voici ce qui vous attend lors de la première connexion:

- Votre compte est automatiquement doté de l'abonnement gratuit **Free**. Vous pouvez télécharger un nombre limité de documents et générer des questions d'examen.
- Votre profil est déjà rempli avec votre nom et votre adresse e-mail (avec Google OAuth, ces données sont directement reprises de Google).
- Vous pouvez commencer immédiatement: téléchargez un document ou générez votre premier examen.

Un aperçu de toutes les fonctionnalités du Dashboard se trouve dans [Comprendre le Dashboard](../user-guide/dashboard.md).

## Mot de passe oublié

Si vous avez oublié votre mot de passe, vous pouvez le réinitialiser via la page de connexion. Cette procédure n'est possible que pour les comptes avec connexion e-mail/mot de passe — les comptes Google OAuth sont gérés via votre compte Google.

1. Ouvrez la page de connexion d'ExamCraft AI.
2. Cliquez sur le lien **Mot de passe oublié?** sous le formulaire de connexion.
3. Entrez l'adresse e-mail avec laquelle vous vous êtes inscrit.
4. Cliquez sur **Envoyer le lien de réinitialisation**.
5. Vous recevrez un e-mail avec un lien pour réinitialiser votre mot de passe. Le lien est valide 1 heure.
6. Cliquez sur le lien dans l'e-mail et entrez votre nouveau mot de passe (au minimum 8 caractères, avec une majuscule et un chiffre/caractère spécial).
7. Confirmez le nouveau mot de passe et cliquez sur **Enregistrer le mot de passe**.
8. Vous êtes redirigé vers la page de connexion et pouvez immédiatement vous connecter avec votre nouveau mot de passe.

## Compte désactivé ou bloqué

Si vous recevez un message lors de la connexion indiquant que votre compte est désactivé ou bloqué, vous ne pouvez pas vous reconnecter vous-même. Dans ce cas, contactez directement l'administrateur de votre institution ou le support ExamCraft AI.

Raisons possibles d'un blocage de compte:

- Trop de tentatives de connexion échouées (blocage temporaire, levé automatiquement après une période d'attente)
- Désactivation manuelle par un administrateur d'institution
- Abonnement expiré ou annulé au niveau de l'institution

Communiquez à l'administrateur votre adresse e-mail enregistrée afin qu'il puisse vérifier votre compte et le réactiver si nécessaire.
