# Schnellstart

Erstellen Sie Ihre erste KI-Prüfung in 5 Minuten.

## Voraussetzungen

- Moderner Browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Internetverbindung
- ExamCraft AI Account

## Schritt 1: Anmelden

1. Öffnen Sie ExamCraft AI in Ihrem Browser
2. Melden Sie sich mit Ihrem Google-Konto oder E-Mail an
3. Nach der Anmeldung sehen Sie das Dashboard

## Schritt 2: Dokument hochladen

1. Klicken Sie auf **Dokumente hochladen** in der Navigation
2. Ziehen Sie ein PDF oder Word-Dokument in den Upload-Bereich (oder klicken Sie **Dateien auswählen**)
3. Warten Sie, bis die Verarbeitung abgeschlossen ist (10–45 Sekunden)

!!! tip "Tipp"
    Strukturierte Dokumente mit klaren Überschriften liefern bessere Prüfungsfragen.

## Schritt 3: Prüfung erstellen

1. Wechseln Sie zum Reiter **RAG-Prüfung erstellen**
2. Wählen Sie Ihre hochgeladenen Dokumente aus
3. Konfigurieren Sie:
    - **Thema/Fokus**: z.B. "Sortieralgorithmen"
    - **Anzahl Fragen**: 5–10 empfohlen
    - **Fragetypen**: Multiple Choice und/oder Offene Fragen
    - **Schwierigkeitsgrad**: Einfach / Mittel / Schwer
4. Klicken Sie **RAG-Prüfung generieren**

## Schritt 4: Ergebnis prüfen

Nach 20–60 Sekunden sehen Sie Ihre generierten Fragen mit:

- Fragetext und Antwortoptionen
- Korrekte Antwort mit Erklärung
- Quellenangaben aus Ihren Dokumenten
- Zuverlässigkeitswert (Confidence Score) — Qualitätsindikator
