# Schnellstart

Erstellen Sie Ihre erste KI-Pruefung in 5 Minuten.

## Voraussetzungen

- Moderner Browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Internetverbindung
- ExamCraft AI Account

## Schritt 1: Anmelden

1. Oeffnen Sie ExamCraft AI in Ihrem Browser
2. Melden Sie sich mit Ihrem Google-Konto oder E-Mail an
3. Nach der Anmeldung sehen Sie das Dashboard

## Schritt 2: Dokument hochladen

1. Klicken Sie auf **Dokumente hochladen** in der Navigation
2. Ziehen Sie ein PDF oder Word-Dokument in den Upload-Bereich (oder klicken Sie **Dateien auswaehlen**)
3. Warten Sie, bis die Verarbeitung abgeschlossen ist (10--45 Sekunden)

!!! tip "Tipp"
    Strukturierte Dokumente mit klaren Ueberschriften liefern bessere Pruefungsfragen.

## Schritt 3: Pruefung erstellen

1. Wechseln Sie zum Tab **RAG-Pruefung erstellen**
2. Waehlen Sie Ihre hochgeladenen Dokumente aus
3. Konfigurieren Sie:
    - **Thema/Fokus**: z.B. "Sortieralgorithmen"
    - **Anzahl Fragen**: 5--10 empfohlen
    - **Fragetypen**: Multiple Choice und/oder Offene Fragen
    - **Schwierigkeitsgrad**: Einfach / Mittel / Schwer
4. Klicken Sie **RAG-Pruefung generieren**

## Schritt 4: Ergebnis pruefen

Nach 20--60 Sekunden sehen Sie Ihre generierten Fragen mit:

- Fragetext und Antwortoptionen
- Korrekte Antwort mit Erklaerung
- Quellenangaben aus Ihren Dokumenten
- Confidence Score (Qualitaetsindikator)
