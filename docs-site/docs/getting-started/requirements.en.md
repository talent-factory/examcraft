# System Requirements

ExamCraft AI is a fully browser-based platform — you don't need to install anything. This section describes what you need for smooth operation.

## Browser

ExamCraft AI supports all modern browsers. Always use the latest available version, as newer versions offer improved security, speed, and compatibility.

| Browser | Minimum Version |
|---------|-----------------|
| Chrome / Edge | 90+ |
| Firefox | 88+ |
| Safari | 14+ |

!!! tip "Recommendation"
    We recommend using **Google Chrome** or **Microsoft Edge** in the latest version. These browsers offer the best compatibility with all ExamCraft AI features, especially for file uploads and document preview.

Make sure JavaScript is enabled. ExamCraft AI requires JavaScript — without it, the platform cannot be used. Browser extensions that block JavaScript (e.g., certain ad blockers) can impair functionality.

## Recommended Screen Resolution

ExamCraft AI is designed for use on desktop, tablet, and mobile devices. However, for the best user experience when creating and reviewing exam questions, we recommend using a desktop.

- **Desktop**: 1920x1080 or higher
- **Tablet**: 1024x768 or higher
- **Mobile**: 375x667 or higher

On mobile devices, all core features are accessible, but working with the exam composer and review queue is more comfortable on larger screens.

## Network

ExamCraft AI requires an active internet connection for all AI functions. Question generation, document processing, and RAG-based search are performed server-side — local offline use is not possible.

**Minimum bandwidth:**

- **Regular use** (generate questions, upload documents): 5 Mbit/s download / 2 Mbit/s upload
- **Recommended**: 25 Mbit/s or more for smooth document processing

**Note for corporate networks and proxies:**

If you use ExamCraft AI in an institution with a corporate firewall or web proxy, make sure access to the ExamCraft AI domain is not blocked. Contact your IT department if you experience connection issues. In some cases, SSL inspection solutions (deep packet inspection) or restrictive proxy configurations are the cause of connection errors.

## Account

You need a personal user account to use ExamCraft AI. The following login methods are supported:

- **Email and Password** — Classical registration with your email address and a self-chosen password. After registration, you will receive a confirmation email.
- **Google OAuth** — Fast login with your existing Google account (recommended). No separate password needed.

In some institutions, registration is managed centrally by an institution administrator. In this case, you will receive an invitation email, which gives you direct access to the platform without needing to register yourself. Contact your instructor or IT administrator if you expect an invitation.

## Subscription

ExamCraft AI is available in four subscription tiers: **Free**, **Starter**, **Professional**, and **Enterprise**. Available features and quotas (e.g., number of documents and questions you can generate per month) depend on your subscription.

New users automatically start with the free **Free** subscription. This includes a limited number of documents and questions per month and is suitable for getting to know the platform.

For more information about subscription tiers and their features, see [Manage Subscription](../user-guide/subscription.md).

---

!!! note "No Installation Required"
    ExamCraft AI is completely browser-based. You don't need to download or install any software. The platform runs directly in your browser — on Windows, macOS, Linux, and mobile operating systems alike.
