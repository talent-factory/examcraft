# Démarrage rapide

Créez votre premier examen AI en 5 minutes.

## Prérequis

- Navigateur moderne (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Connexion Internet
- Compte ExamCraft AI

## Étape 1: Connexion

1. Ouvrez ExamCraft AI dans votre navigateur
2. Connectez-vous avec votre compte Google ou votre adresse e-mail
3. Après la connexion, vous voyez le Dashboard

## Étape 2: Télécharger un document

1. Cliquez sur **Télécharger des documents** dans la navigation
2. Glissez-déposez un PDF ou un document Word dans la zone de téléchargement (ou cliquez sur **Sélectionner des fichiers**)
3. Attendez que le traitement soit terminé (10–45 secondes)

!!! tip "Conseil"
    Les documents structurés avec des titres clairs produisent de meilleures questions d'examen.

## Étape 3: Créer un examen

1. Allez à l'onglet **Créer un examen RAG**
2. Sélectionnez vos documents téléchargés
3. Configurez:
    - **Sujet/Domaine**: par ex. "Algorithmes de tri"
    - **Nombre de questions**: 5–10 recommandé
    - **Types de questions**: Choix multiples et/ou Questions ouvertes
    - **Niveau de difficulté**: Facile / Moyen / Difficile
4. Cliquez sur **Générer l'examen RAG**

## Étape 4: Vérifier les résultats

Après 20–60 secondes, vous voyez vos questions générées avec:

- Texte de la question et options de réponse
- Réponse correcte avec explication
- Sources de vos documents
- Indice de confiance (Confidence Score) — indicateur de qualité
