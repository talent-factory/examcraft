# Dashboard

Le Dashboard est la page d'accueil d'ExamCraft AI après la connexion. Il offre un aperçu rapide de vos activités et un accès direct à toutes les fonctionnalités principales de la plateforme.

<!-- screenshot: dashboard-overview.png -->

## Statistiques

En haut du Dashboard, vous voyez quatre indicateurs d'un coup d'œil:

| Statistique | Description |
|-----------|-------------|
| Questions générées | Nombre total de questions créées par l'IA jusqu'à présent |
| Documents | Nombre de documents téléchargés et traités avec succès |
| Questions validées | Nombre de questions approuvées dans la Review Queue |
| Examens | Nombre d'examens créés dans Exam Composer |

Ces chiffres se mettent à jour en temps réel et vous aident à garder une vue d'ensemble de votre travail.

<!-- screenshot: dashboard-statistics.png -->

## Tuiles d'accès rapide

Les tuiles d'accès rapide permettent un accès direct aux domaines les plus importants de la plateforme:

| Tuile | Destination | Description |
|--------|------|-------------|
| Télécharger des documents | [Documents](documents.md) | Télécharger et traiter de nouveaux matériels d'apprentissage |
| Créer un examen | [Créer un examen](exam-create.md) | Générer des questions IA sans document source |
| Créer un examen RAG | [Examen RAG](rag-exam.md) | Générer des questions à partir de vos documents |
| Réviser les questions | [Review Queue](review-queue.md) | Vérifier et approuver les questions générées |
| Assembler un examen | [Exam Composer](exam-composer.md) | Assembler les questions approuvées en examens |
| Bibliothèque de prompts | [Bibliothèque de prompts](prompt-library.md) | Gérer les prompts AI réutilisables |

Cliquez sur une tuile pour accéder directement au domaine souhaité.

## Activités récentes

La section «Activités récentes» montre vos questions générées récemment et vos documents téléchargés récemment. Cela vous permet de reprendre rapidement les tâches interrompues.

- **Questions générées récemment**: Cliquez sur une question pour l'ouvrir dans la Review Queue
- **Documents téléchargés récemment**: Cliquez sur un document pour l'ouvrir dans la bibliothèque de documents

<!-- screenshot: dashboard-recent-activity.png -->

## Badge d'abonnement

En haut à gauche du Dashboard, vous voyez un badge coloré avec votre abonnement actuel, par exemple «Free», «Starter» ou «Professional». Ce badge vous montre à tout moment les fonctionnalités qui vous sont disponibles.

Cliquez sur le badge pour accéder à la [page d'abonnement](subscription.md) et consulter ou mettre à niveau votre plan.

!!! tip "Conseil: Dashboard comme point de départ"
    Le Dashboard vous donne un aperçu rapide. Pour un flux de travail efficace, nous recommandons: d'abord télécharger des documents, puis générer des questions, traiter la Review Queue et enfin assembler l'examen dans Exam Composer.
