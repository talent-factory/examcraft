# Troubleshooting

## Upload Issues

**Symptom:** "Upload failed"

1. Check your internet connection
2. Make sure the file is smaller than 50 MB
3. Check the format (PDF, DOC, DOCX, MD, TXT)
4. Clear your browser cache
5. Try a different browser

## Generation Errors

**Symptom:** "Error during generation"

1. Wait 30 seconds and try again
2. Reduce the number of questions
3. Simplify the topic
4. Check the system status

## Performance Issues

**Symptom:** Slow loading times

1. Close other browser tabs
2. Clear your browser cache
3. Update your browser
4. Check your internet speed

## Display Errors

**Symptom:** Elements not displayed correctly

1. Refresh the page (F5)
2. Hard refresh (Ctrl+Shift+R)
3. Check your browser version
4. Disable browser extensions

## Review Queue and Exam Composer

**Questions do not appear in the Exam Composer**

**Cause**: Only questions with status "Approved" appear in the Exam Composer.

**Solution**:
1. Open the [Review Queue](../user-guide/review-queue.md)
2. Filter by status "Pending"
3. Review and approve the desired questions
4. They will then be immediately available in the Exam Composer

## Document Chat

**"Document Chat" is not visible or accessible**

**Cause**: The Document Chat is available from the Starter subscription.

**Solution**: Check your current subscription under [Subscription](../user-guide/subscription.md).
This feature is not included in the Free plan.

## Build and Documentation

**The mkdocs build fails with missing files**

**Cause**: Pages referenced in `mkdocs.yml` do not yet exist.

**Solution**: Make sure all `.md` files listed in the navigation actually exist.
