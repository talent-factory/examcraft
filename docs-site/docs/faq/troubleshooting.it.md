# Risoluzione dei problemi

## Problemi di caricamento

**Sintomo:** "Caricamento non riuscito"

1. Verificate la vostra connessione Internet
2. Assicuratevi che il file sia più piccolo di 50 MB
3. Controllate il formato (PDF, DOC, DOCX, MD, TXT)
4. Svuotate la cache del browser
5. Provate un browser diverso

## Errori di generazione

**Sintomo:** "Errore nella generazione"

1. Attendete 30 secondi e riprovate
2. Riducete il numero di domande
3. Semplificate l'argomento
4. Verificate lo stato del sistema

## Problemi di performance

**Sintomo:** Tempi di caricamento lenti

1. Chiudete altri tab del browser
2. Svuotate la cache del browser
3. Aggiornate il vostro browser
4. Verificate la vostra velocità Internet

## Errori di visualizzazione

**Sintomo:** Gli elementi non vengono visualizzati correttamente

1. Aggiornate la pagina (F5)
2. Hard-Refresh (Ctrl+Shift+R)
3. Verificate la versione del browser
4. Disabilitate le estensioni del browser

## Review Queue e Exam Composer

## Le domande non appaiono in Exam Composer

**Causa**: Solo le domande con stato "Approvata" appaiono in Exam Composer.

**Soluzione**:
1. Aprite la [Review Queue](../user-guide/review-queue.md)
2. Filtrate per stato "In sospeso"
3. Verificate e approvate le domande desiderate
4. Saranno immediatamente disponibili in Exam Composer

## Chat Documento

## "Chat Documento" non è visibile o non è accessibile

**Causa**: Chat Documento è disponibile da abbonamento Starter.

**Soluzione**: Verificate il vostro abbonamento attuale sotto [Abbonamento](../user-guide/subscription.md).
Con il piano Free questa funzione non è inclusa.

## Build e documentazione

## Il mkdocs build fallisce con file mancanti

**Causa**: Le pagine referenziate in `mkdocs.yml` non esistono ancora.

**Soluzione**: Assicuratevi che tutti i file `.md` registrati nella navigazione
siano effettivamente presenti.
