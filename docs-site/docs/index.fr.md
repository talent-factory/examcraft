# Bienvenue sur ExamCraft AI

ExamCraft AI est une plateforme basée sur l'IA pour générer automatiquement des questions d'examen à partir de vos matériels de cours.

## Démarrage rapide

Créez votre premier examen AI en 5 minutes:

1. **Télécharger un document** – Chargez un PDF ou un document Word
2. **Configurer l'examen** – Sélectionnez le type de question, le niveau de difficulté et le nombre
3. **Générer** – L'IA crée des questions basées sur vos matériels
4. **Réviser et exporter** – Vérifiez et exportez les résultats
