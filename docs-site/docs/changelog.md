# Changelog

Änderungen und Neuigkeiten zu ExamCraft AI.

## 2026

## Version 2.0 — April 2026

### Neu

- **Dashboard**: Neue Startseite mit Statistiken, Schnellzugriff-Kacheln und letzten Aktivitäten
- **Review Queue**: Manuelle Prüfung und Freigabe von KI-generierten Fragen vor der Prüfungszusammenstellung
- **Prüfungskomponist**: Genehmigte Fragen zu vollständigen Prüfungen zusammenstellen und exportieren (PDF, Word, JSON)
- **Prompt-Bibliothek**: Neu für alle DOZENT- und ADMIN-Benutzer zugänglich (bisher nur Admin-Panel)
- **Profilseite**: Persönliche Daten und Passwort direkt im Benutzerprofil verwalten
- **Abonnementseite**: Übersicht, Upgrade und Rechnungsverwaltung über Stripe-Portal

### Verbessert

- Vollständige Endbenutzer-Dokumentation auf [docs.examcraft.ch](https://docs.examcraft.ch)
- Workflow Fragengenerierung → Review Queue → Prüfungskomponist als roter Faden durch die Plattform

### März 2026

- Smart Help Widget und Docs-Site (in Entwicklung)
- Internationalisierung (i18n) Infrastruktur
- Auto Composition Engine
