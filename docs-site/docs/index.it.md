## Avvio rapido

Create il vostro primo esame con intelligenza artificiale in 5 minuti:

1. **Caricamento documento** – Caricate un file PDF o Word
2. **Configurazione esame** – Scegliete il tipo di domanda, livello di difficoltà e numero
3. **Generazione** – L'IA crea le domande in base ai vostri materiali
4. **Revisione ed esportazione** – Verificate ed esportate i risultati
