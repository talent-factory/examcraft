## Endpoints Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/documents` | Upload document |
| GET | `/api/v1/documents` | List documents |
| GET | `/api/v1/documents/{id}` | Get document details |
| DELETE | `/api/v1/documents/{id}` | Delete document |
| GET | `/api/v1/documents/{id}/chunks` | Get document chunks |

## Upload Document

```http
POST /api/v1/documents
```

<ParamField path="file" type="file" required>
  Document file (PDF, DOCX, MD, TXT)
</ParamField>

<ParamField path="tags" type="string">
  Comma-separated tags
</ParamField>

<ParamField path="language" type="string">
  Document language (de, en)
</ParamField>

### Request

<CodeGroup>

```python Python
import requests

with open("textbook.pdf", "rb") as f:
    response = requests.post(
        "https://api.examcraft.ai/api/v1/documents",
        headers={"Authorization": f"Bearer {token}"},
        files={"file": f},
        data={
            "tags": "algorithms,data-structures",
            "language": "de"
        }
    )

document = response.json()
```

```javascript JavaScript
const formData = new FormData();
formData.append('file', fileInput.files[0]);
formData.append('tags', 'algorithms,data-structures');
formData.append('language', 'de');

const response = await fetch(
  'https://api.examcraft.ai/api/v1/documents',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`
    },
    body: formData
  }
);

const document = await response.json();
```

```bash cURL
curl -X POST https://api.examcraft.ai/api/v1/documents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@textbook.pdf" \
  -F "tags=algorithms,data-structures" \
  -F "language=de"
```

</CodeGroup>

### Response

```json
{
  "id": 123,
  "filename": "textbook.pdf",
  "size": 2456789,
  "format": "pdf",
  "processing_status": "processing",
  "language": "de",
  "tags": ["algorithms", "data-structures"],
  "uploaded_at": "2025-01-08T14:32:00Z",
  "user_id": 1,
  "institution_id": 1
}
```

## List Documents

```http
GET /api/v1/documents
```

<ParamField query="limit" type="number" default="20">
  Number of results (max 100)
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor
</ParamField>

<ParamField query="tags" type="string">
  Filter by tags (comma-separated)
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.get(
    "https://api.examcraft.ai/api/v1/documents",
    headers={"Authorization": f"Bearer {token}"},
    params={
        "limit": 20,
        "tags": "algorithms"
    }
)

documents = response.json()["data"]
```

```javascript JavaScript
const response = await fetch(
  'https://api.examcraft.ai/api/v1/documents?limit=20&tags=algorithms',
  {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const { data: documents } = await response.json();
```

</CodeGroup>

### Response

```json
{
  "data": [
    {
      "id": 123,
      "filename": "textbook.pdf",
      "processing_status": "ready",
      "uploaded_at": "2025-01-08T14:32:00Z"
    }
  ],
  "pagination": {
    "next_cursor": "eyJpZCI6MTIzfQ==",
    "has_more": true,
    "total": 156
  }
}
```


<CodeGroup>

```python Python
response = requests.get(
    f"https://api.examcraft.ai/api/v1/documents/{document_id}",
```

```javascript JavaScript
const response = await fetch(
```

## Delete Document

```http
DELETE /api/v1/documents/{id}
```

### Request

<CodeGroup>

```python Python
response = requests.delete(
    f"https://api.examcraft.ai/api/v1/documents/{document_id}",
    headers={"Authorization": f"Bearer {token}"}
)
```

```javascript JavaScript
await fetch(
  `https://api.examcraft.ai/api/v1/documents/${documentId}`,
  {
    method: 'DELETE',
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);
```

</CodeGroup>

### Response

```json
{
  "success": true,
  "message": "Document deleted successfully"
}
```

<Warning>
  Gelöschte Dokumente können nicht wiederhergestellt werden!
</Warning>

## Get Document Chunks

```http
GET /api/v1/documents/{id}/chunks
```

Returns all semantic chunks of a document.

### Request

<CodeGroup>

```python Python
response = requests.get(
    f"https://api.examcraft.ai/api/v1/documents/{document_id}/chunks",
    headers={"Authorization": f"Bearer {token}"}
)

chunks = response.json()["chunks"]
```

```javascript JavaScript
const response = await fetch(
  `https://api.examcraft.ai/api/v1/documents/${documentId}/chunks`,
  {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
);

const { chunks } = await response.json();
```

</CodeGroup>

### Response

```json
{
  "chunks": [
    {
      "id": 1,
      "chunk_index": 0,
      "content": "Heapsort is a comparison-based sorting algorithm...",
      "metadata": {
        "section_title": "Sorting Algorithms",
        "page_number": 42,
        "chunk_size": 487
      }
    }
  ],
  "total": 342
}
```

## Processing Status

| Status | Description | Duration |
|--------|-------------|----------|
| `uploading` | File is being uploaded | Depends on size |
| `processing` | Docling is processing | 30s - 5min |
| `chunking` | Semantic chunking | 10s - 1min |
| `embedding` | Creating embeddings | 5s - 30s |
| `indexing` | Qdrant indexing | 5s - 20s |
| `ready` | Ready for RAG | - |
| `failed` | Error occurred | - |
