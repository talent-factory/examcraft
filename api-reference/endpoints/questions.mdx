## Endpoints Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/questions/generate` | Generate questions (topic-based) |
| POST | `/api/v1/questions/generate-rag` | Generate questions (RAG-based) |
| GET | `/api/v1/questions` | List questions |
| GET | `/api/v1/questions/{id}` | Get question details |
| PUT | `/api/v1/questions/{id}` | Update question |
| DELETE | `/api/v1/questions/{id}` | Delete question |

## Generate Questions (Topic-Based)

```http
POST /api/v1/questions/generate
```

Generate questions without documents.

### Request Body

<ParamField body="topic" type="string" required>
  Topic for question generation
</ParamField>

<ParamField body="difficulty" type="string" required>
  easy, medium, hard
</ParamField>

<ParamField body="num_questions" type="number" required>
  Number of questions (1-20)
</ParamField>

<ParamField body="question_types" type="array" required>
  ["multiple_choice", "open_ended", "code"]
</ParamField>

<ParamField body="language" type="string" default="de">
  de, en
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.post(
    "https://api.examcraft.ai/api/v1/questions/generate",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "topic": "Python Lists - Methods and Operations",
        "difficulty": "medium",
        "num_questions": 5,
        "question_types": ["multiple_choice", "open_ended"],
        "language": "de"
    }
)

questions = response.json()["questions"]
```

```javascript JavaScript
const response = await fetch(
  'https://api.examcraft.ai/api/v1/questions/generate',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      topic: 'Python Lists - Methods and Operations',
      difficulty: 'medium',
      num_questions: 5,
      question_types: ['multiple_choice', 'open_ended'],
      language: 'de'
    })
  }
);

const { questions } = await response.json();
```

</CodeGroup>

### Response

```json
      "id": 1,

  ],
  "metadata": {
    "topic": "Python Lists - Methods and Operations",

  }
}
```

## Generate Questions (RAG-Based)

```http
POST /api/v1/questions/generate-rag
```

Generate questions from documents using RAG.

### Request Body

<ParamField body="document_ids" type="array" required>
  Document IDs to use [123, 124]
</ParamField>

<ParamField body="topic" type="string">
  Optional focus topic
</ParamField>

<ParamField body="difficulty" type="string" required>
  easy, medium, hard
</ParamField>

<ParamField body="num_questions" type="number" required>
  Number of questions (1-20)
</ParamField>

<ParamField body="question_types" type="array" required>
  ["multiple_choice", "open_ended"]
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.post(
    "https://api.examcraft.ai/api/v1/questions/generate-rag",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "document_ids": [123, 124],
        "topic": "Sorting Algorithms Complexity",
        "difficulty": "medium",
        "num_questions": 5,
        "question_types": ["multiple_choice", "open_ended"]
    }
)

questions = response.json()["questions"]

for q in questions:
    print(f"Q: {q['question']}")
    print(f"Confidence: {q['confidence']}")
    print(f"Sources: {len(q['sources'])}")
```

```javascript JavaScript
const response = await fetch(
  'https://api.examcraft.ai/api/v1/questions/generate-rag',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      document_ids: [123, 124],
      topic: 'Sorting Algorithms Complexity',
      difficulty: 'medium',
      num_questions: 5,
      question_types: ['multiple_choice', 'open_ended']
    })
  }
);

const { questions } = await response.json();
```

</CodeGroup>

### Response

```json
{

      "type": "multiple_choice",

      "correct_answer": 1,
      "explanation": "Heapsort hat garantierte O(n log n)...",
      "bloom_level": 2,
      "difficulty": 2,
      "confidence": 0.95,
      "sources": [
        {
          "document_id": 123,
          "document_name": "algorithms.pdf",

```

## List Questions

```http
GET /api/v1/questions
```

### Query Parameters

<ParamField query="limit" type="number" default="20">
  Results per page (max 100)
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor
</ParamField>

<ParamField query="type" type="string">
  Filter by type (multiple_choice, open_ended, code)
</ParamField>

<ParamField query="difficulty" type="string">
  Filter by difficulty (easy, medium, hard)
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.get(
    "https://api.examcraft.ai/api/v1/questions",
    headers={"Authorization": f"Bearer {token}"},
    params={
        "limit": 20,
        "type": "multiple_choice",
        "difficulty": "medium"
    }
)

questions = response.json()["data"]
```

</CodeGroup>

### Response

```json
{
  "data": [
    {
      "id": 1,
      "type": "multiple_choice",
      "question": "Was ist die Zeitkomplexität...",
      "created_at": "2025-01-08T14:32:00Z"
    }
  ],
  "pagination": {
    "next_cursor": "eyJpZCI6MX0=",
    "has_more": true,
    "total": 45
  }
}
```

## Update Question

```http
PUT /api/v1/questions/{id}
```

### Request

<CodeGroup>

```python Python
response = requests.put(
    f"https://api.examcraft.ai/api/v1/questions/{question_id}",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "question": "Updated question text",
        "explanation": "Updated explanation"
    }
)
```

</CodeGroup>

## Delete Question

```http
DELETE /api/v1/questions/{id}
```

<Warning>
  Deleted questions cannot be recovered!
</Warning>
