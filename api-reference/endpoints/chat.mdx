## Endpoints Overview

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/chat/sessions` | Create chat session |
| GET | `/api/v1/chat/sessions` | List sessions |
| GET | `/api/v1/chat/sessions/{id}` | Get session details |
| POST | `/api/v1/chat/message` | Send message |
| POST | `/api/v1/chat/sessions/{id}/to-document` | Export to document |
| GET | `/api/v1/chat/sessions/{id}/download` | Download as Markdown |
| DELETE | `/api/v1/chat/sessions/{id}` | Delete session |

## Create Chat Session

```http
POST /api/v1/chat/sessions
```

### Request Body

<ParamField body="document_id" type="number" required>
  Document to chat with
</ParamField>

<ParamField body="title" type="string">
  Optional session title
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.post(
    "https://api.examcraft.ai/api/v1/chat/sessions",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "document_id": 123,
        "title": "Heapsort Discussion"
    }
)

session = response.json()
session_id = session["id"]
```

```javascript JavaScript
const response = await fetch(
  'https://api.examcraft.ai/api/v1/chat/sessions',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      document_id: 123,
      title: 'Heapsort Discussion'
    })
  }
);

const { id: sessionId } = await response.json();
```

</CodeGroup>

### Response

```json
{
  "id": 1,
  "document_id": 123,
  "title": "Heapsort Discussion",
  "created_at": "2025-01-08T14:32:00Z",
  "message_count": 0
}
```

## Send Message

```http
POST /api/v1/chat/message
```

### Request Body

<ParamField body="session_id" type="number" required>
  Chat session ID
</ParamField>

<ParamField body="message" type="string" required>
  User message
</ParamField>

### Request

<CodeGroup>

```python Python
response = requests.post(
    "https://api.examcraft.ai/api/v1/chat/message",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "session_id": session_id,
        "message": "Erkläre mir den Heapsort Algorithmus"
    }
)

answer = response.json()
print(f"Bot: {answer['response']}")
print(f"Confidence: {answer['confidence']}")
print(f"Sources: {len(answer['sources'])}")
```

```javascript JavaScript
const response = await fetch(
  'https://api.examcraft.ai/api/v1/chat/message',
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      session_id: sessionId,
      message: 'Erkläre mir den Heapsort Algorithmus'
    })
  }
);

const answer = await response.json();
```

</CodeGroup>

### Response

```json
{
  "response": "Heapsort ist ein vergleichsbasierter Sortieralgorithmus...",
  "confidence": 0.95,
  "sources": [
    {
      "document_id": 123,
      "document_name": "algorithms.pdf",
      "page_number": 42,
      "chunk_text": "Heapsort has a time complexity...",
      "relevance_score": 0.95
    }
  ],
  "message_id": 1,
  "timestamp": "2025-01-08T14:32:05Z"
}
```

## Get Session Details

```http
GET /api/v1/chat/sessions/{id}
```

Returns session with full message history.

### Request

<CodeGroup>

```python Python
response = requests.get(
    f"https://api.examcraft.ai/api/v1/chat/sessions/{session_id}",
    headers={"Authorization": f"Bearer {token}"}
)

session = response.json()
messages = session["messages"]
```

</CodeGroup>

### Response

```json
{
  "id": 1,
  "document_id": 123,
  "title": "Heapsort Discussion",
  "created_at": "2025-01-08T14:32:00Z",
  "messages": [
    {
      "id": 1,
      "role": "user",
      "content": "Erkläre mir den Heapsort Algorithmus",
      "timestamp": "2025-01-08T14:32:05Z"
    },
    {
      "id": 2,
      "role": "assistant",
      "content": "Heapsort ist ein vergleichsbasierter...",
      "confidence": 0.95,
      "sources": [...],
      "timestamp": "2025-01-08T14:32:08Z"
    }
  ]
}
```

## Export Chat to Document

```http
POST /api/v1/chat/sessions/{id}/to-document
```

Saves chat as document in library.

### Request

<CodeGroup>

```python Python
response = requests.post(
    f"https://api.examcraft.ai/api/v1/chat/sessions/{session_id}/to-document",
    headers={"Authorization": f"Bearer {token}"}
)

document_id = response.json()["document_id"]
```

</CodeGroup>

### Response

```json
{
  "document_id": 456,
  "message": "Chat exported to document successfully"
}
```

## Download Chat as Markdown

```http
GET /api/v1/chat/sessions/{id}/download
```

Returns Markdown file.

### Request

<CodeGroup>

```python Python
response = requests.get(
    f"https://api.examcraft.ai/api/v1/chat/sessions/{session_id}/download",
    headers={"Authorization": f"Bearer {token}"}
)

with open("chat_export.md", "w") as f:
    f.write(response.text)
```

</CodeGroup>
